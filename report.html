Project 2: Web Service | Report

Here's a general overview of using this web service:

Endpoints:

  • GET /photos/:month/:day - returns photos found on the given month and day. Entries are ordered by year
  • POST /photos - adds a word with the given identification options (year, month, day, image name, image source/link, and the image description
  • DELETE /photos/:id - removes a photo based on the given ID (HARD DELETE)
